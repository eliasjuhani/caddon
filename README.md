#Collect addon

Chrome/Edge-laajennus Collect@Store-tilausten reaaliaikaiseen seurantaan.

Toiminnallisuus:
    Laajennus tarkkailee SAP Launchpadia taustalla ja ilmoittaa uusista tilauksista koko ruudun hälytyksellä, jota käyttäjä voi muokata suoraan selaimesta haluamansa näköiseksi.

Jos et osaa ottaa käyttöön:
    Lataa projektin tiedostot, siirry selaimessa osoitteeseen edge://extensions, ota kehittäjätila käyttöön ja valitse "Load unpacked".
